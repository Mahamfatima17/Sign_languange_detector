# Models Directory

This directory stores trained models for the ASL Translation System.

## Files

| File | Pipeline | Generated By |
|------|----------|-------------|
| `letter_model.pkl` | Random Forest (letters) | `python train_letters.py` |
| `asl_model.h5` | LSTM (dynamic signs) | `python train_model.py` |

## Training Instructions

### Letter Model (Random Forest)
```bash
python collect_letters.py    # Collect letter data
python train_letters.py      # Train and save model
```

### Dynamic Sign Model (LSTM)
```bash
python collect_data.py       # Collect 30-frame sequences
python train_model.py        # Train and save model
```

## Model Architecture

### LSTM Model
```
Input (30, 1662)
  → LSTM(64, return_sequences=True)
  → Dropout(0.2)
  → LSTM(128)
  → Dropout(0.3)
  → Dense(64, relu)
  → Dropout(0.2)
  → Dense(num_classes, softmax)
```

### Random Forest
- 200 trees, max_depth=20
- Features: 63 per sample (21 landmarks × 3 coordinates)

## Supported Formats
- `.h5` — Keras HDF5 format (LSTM)
- `.pkl` — Pickle format (Random Forest)
- SavedModel directory — TensorFlow SavedModel format
